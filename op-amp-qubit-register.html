Chapter 17: Active Op-Amp Implementation of the Parallel Analog Register

A classical parallel analog search system can be physically realized at room temperature using commercially available, low-noise operational amplifiers and passive components. By mapping the \(n\)-qubit register to a network of \(N = 2^n\) coupled active RC resonators, we can execute the amplitude-amplification algorithm purely through classical wave resonance and spatial summing.

This chapter details the circuit design for a 4-channel (\(N=4\), representing a 2-qubit register) analog search engine.

17.1 Circuit Architecture & Component Selection

To minimize phase jitter and thermal drift—which limit the resolution of the analog register—the circuit is designed with high-precision components: * Operational Amplifiers: The Texas Instruments OPA1612 (dual) or OPA1611 (single) is selected. It features an ultra-low input voltage noise density of \(1.1\text{ nV}/\sqrt{\text{Hz}}\) at \(1\text{ kHz}\), ultra-low total harmonic distortion (THD) of \(0.000015\%\), and a high gain-bandwidth product of \(40\text{ MHz}\). This minimizes phase errors at the resonant frequency. * Resistors: \(0.1\%\) tolerance, low-temperature-coefficient (\(25\text{ ppm/}^\circ\text{C}\)) thin-film metal resistors to ensure precise matching of the resonant frequencies and coupling coefficients across all channels. * Capacitors: \(1\%\) tolerance COG/NPO ceramic capacitors to eliminate dielectric absorption and minimize thermal drift.

17.2 The Active Resonator Node (Tow-Thomas Integrator Loop)

Each of the 4 nodes is implemented as an active Tow-Thomas resonator, which behaves as a second-order, high-Q classical harmonic oscillator. A single node \(j \in \{1, 2, 3, 4\}\) utilizes three op-amp stages (one OPA1612 and half of an OPA1611):

  1. Stage 1: Summing Lossy Integrator. Integrates the sum of the feedback and coupling inputs.
    • Feedback capacitor: \(C_1 = 1\text{ nF}\)
    • Damping resistor: \(R_D = 1\text{ M}\Omega\) (establishes a very high quality factor, \(Q \approx 63\))
    • Integration resistor: \(R = 15.9\text{ k}\Omega\)
  2. Stage 2: Pure Integrator. Integrates the output of Stage 1.
    • Feedback capacitor: \(C_2 = 1\text{ nF}\)
    • Integration resistor: \(R = 15.9\text{ k}\Omega\)
  3. Stage 3: Unity-Gain Inverter. Inverts the output of Stage 2 to restore positive phase feedback.
    • Input and feedback resistors: \(R_{\text{inv}} = 10\text{ k}\Omega\)

Design Equations:

The nominal resonant frequency \(f_0\) for each node is defined by: \[f_0 = \frac{1}{2\pi R C} = \frac{1}{2\pi \left(15.9\text{ k}\Omega\right) \left(1\text{ nF}\right)} \approx 10.01\text{ kHz}\]

This frequency is chosen to keep parasitic capacitances negligible and to ensure the op-amp remains well within its open-loop gain margin (\(130\text{ dB}\) at \(10\text{ kHz}\)).

17.3 The Global Summing Bus & Coupling Network

To implement the continuous-time Grover reflection-about-the-mean, a global summing bus continuously calculates the average state voltage of the 4 nodes.

The Summing Circuit:

A single OPA1612 is configured as a two-stage summing amplifier: * Stage A (Inverting Sum): The outputs \(V_1, V_2, V_3, V_4\) from each of the 4 resonators are routed through matching resistors \(R_{\text{in}} = 10\text{ k}\Omega\) to the inverting input of the op-amp. The feedback resistor is set to \(R_{\text{fb1}} = 2.5\text{ k}\Omega\). The output is: \[V_{\text{inv\_sum}} = -\sum_{k=1}^4 \left( \frac{R_{\text{fb1}}}{R_{\text{in}}} V_k \right) = -0.25 \sum_{k=1}^4 V_k = -V_{\text{mean}}\] * Stage B (Inverter Buffer): Inverts \(-V_{\text{mean}}\) using a unity-gain inverting amplifier (\(R = 10\text{ k}\Omega\)) to generate the positive mean voltage \(+V_{\text{mean}}\).

The Coupling Term Injection:

The feedback coupling term \(\gamma \left(V_{\text{mean}} - V_j\right)\) is injected directly into the summing junction of the first integrator of Resonator \(j\). This is achieved by running two parallel input resistors to the virtual ground summing node of Stage 1: 1. A resistor \(R_{\text{coup}} = 31.8\text{ k}\Omega\) connected to \(+V_{\text{mean}}\). 2. A resistor \(R_{\text{self}} = 31.8\text{ k}\Omega\) connected to the resonator’s own inverted output \(V_{\text{inv}, j} = -V_j\).

Because the summing junction of the integrator is inverting, the net current injected into the feedback capacitor is: \[I_j = C \frac{dV_{\text{int}}}{dt} = \frac{V_{\text{mean}}}{R_{\text{coup}}} + \frac{V_{\text{inv}, j}}{R_{\text{self}}} = \frac{1}{R_{\text{coup}}} \left( V_{\text{mean}} - V_j \right)\]

This layout implements the subtraction and coupling scaling \(\gamma = \frac{R}{R_{\text{coup}}} = 0.5\) using only two passive resistors per node, preventing the noise and offset voltages of additional active difference amplifiers.

17.4 The Active Oracle Phase Switch

To mark the target item in the database, the oracle must invert the phase of the target resonator’s output (\(V_{\text{target}} \to -V_{\text{target}}\)).

This is implemented using a high-speed, low-noise SPDT analog switch, such as the Analog Devices ADG1419 (on-resistance \(R_{\text{on}} \approx 2.1 \ \Omega\)). Each resonator channel is equipped with one switch: * Input S1: Connected to the normal in-phase output \(+V_j\). * Input S2: Connected to the inverted output \(V_{\text{inv}, j} = -V_j\). * Drain (Output): Connected to the summing bus input resistor \(R_{\text{in}}\) and the coupling resistor \(R_{\text{self}}\). * Logic Control Pin: Connected to a digital control bus (e.g., from a microcontroller or timer).

Under default conditions, all four analog switches are set to route \(+V_j\). When the oracle is activated to target node \(k\), the control line for switch \(k\) is pulled high, toggling its output to \(-V_k\). This introduces an instantaneous phase shift of exactly \(\pi\) radians (\(180^\circ\)) to the target node’s contribution to the summing network.

17.5 Operational Protocol

The register operates in three distinct, sequentially timed phases:

  1. Initialization Phase (State Superposition): A brief \(10\text{ kHz}\) sinusoidal calibration signal \(V_{\text{init}}(t) = V_0 \sin(\omega_0 t)\) is injected into the global summing bus. This external drive forces all 4 active resonators to phase-lock and oscillate with identical amplitudes and phases, establishing the uniform initial state: \[V_1(0) = V_2(0) = V_3(0) = V_4(0) = \frac{1}{2} V_0\]
  2. Oracle Execution Phase: The analog switch for the target index (e.g., Channel 3) is toggled to its inverted position. The phase-inverted signal \(-V_3\) is now injected into the global summing bus.
  3. Amplitude Amplification Phase (Analog Evolution): The system is allowed to evolve freely under the influence of the global coupling bus. The phase mismatch between the target node and the remaining nodes drives a coherent energy transfer.
    • The voltage amplitude \(|V_3(t)|\) of the target channel grows constructively.
    • The voltage amplitudes of the non-target channels (\(|V_1(t)|\), \(|V_2(t)|\), \(|V_4(t)|\)) decay destructively.
  4. Readout: The optimal evolution interval is governed by the beat frequency of the coupled system, matching the analytical limit: \[t_{\text{opt}} \approx \frac{\pi}{4}\sqrt{N} \cdot \frac{1}{f_0} = \frac{\pi}{4}\sqrt{4} \cdot \frac{1}{10\text{ kHz}} \approx 157\text{ }\mu\text{s}\] At \(t_{\text{opt}}\), the envelope voltages of the four channels are read using simple op-amp diode peak detectors. The channel with the highest DC voltage represents the target index.

17.6 Verification of Noise Limits on Register Scaling

To evaluate the maximum scaling capacity (\(n_{\text{max}}\)) of this room-temperature op-amp register, we calculate the signal-to-noise ratio (SNR) at the summing junction.

The total root-mean-square (RMS) thermal noise voltage \(V_n\) generated by the low-noise OPA1612 summing amplifier over a measurement bandwidth \(B = 100\text{ kHz}\) is governed by the input voltage noise density \(e_n = 1.1\text{ nV}/\sqrt{\text{Hz}}\) and the thermal noise of the input resistors \(R_{\text{in}} = 10\text{ k}\Omega\):

\[V_{n, \text{resistor}} = \sqrt{4 k_B T R_{\text{in}} B} = \sqrt{4 \left(1.38 \times 10^{-23}\right) \left(298\right) \left(10\text{ k}\Omega\right) \left(100\text{ kHz}\right)} \approx 406\text{ nV}\] \[V_{n, \text{opamp}} = e_n \sqrt{B} = \left(1.1\text{ nV}/\sqrt{\text{Hz}}\right) \sqrt{100\text{ kHz}} \approx 348\text{ nV}\] \[V_{n, \text{total}} = \sqrt{V_{n, \text{resistor}}^2 + V_{n, \text{opamp}}^2} \approx 535\text{ nV}\]

As the database size \(N = 2^n\) increases, the peak signal amplitude of the target node scales as \(1/\sqrt{N}\) relative to the initial drive voltage \(V_0 \approx 5\text{ V}\):

\[V_{\text{target, peak}} = \frac{V_0}{\sqrt{N}}\]

To resolve the target node with a minimum acceptable SNR of \(20\text{ dB}\) (a factor of 10 in voltage amplitude above the noise floor):

\[\frac{V_{\text{target, peak}}}{V_{n, \text{total}}} \ge 10 \implies \frac{V_0}{\sqrt{N}} \ge 10 \cdot V_{n, \text{total}}\] \[\sqrt{N} \le \frac{V_0}{10 \cdot V_{n, \text{total}}} = \frac{5.0\text{ V}}{10 \cdot \left(535 \times 10^{-9}\text{ V}\right)} \approx 934,579\] \[N \le 8.7 \times 10^{11} \implies n \le 39\text{ qubits}\]

This demonstrates that when using commercial low-noise op-amps like the OPA1612, room-temperature thermal noise and op-amp voltage noise do not impose a practical limitation on the register’s precision for scales up to approximately 39 equivalent qubits. Instead, scaling limits are dominated by the physical requirement to build and match \(N = 2^n\) individual physical RC resonator stages.

Chapter 18: Full System Circuit Design for the Analog Resonant Search

We are implementing Grover’s algorithm. To execute the entire search process, the active resonator nodes and global summing bus are integrated with timing, control, reset, and peak-detection sub-circuits [Chapter 17]. This system physically maps the complex vector space of logical propositions [Chapter 13] onto a multichannel classical wave-resonance space, where each physical resonator represents the probability amplitude and precessional phase of an elementary proposition.

18.1 Component Blocks

  1. Coupled Active Resonator Array: A set of \(N = 2^n\) identical Tow-Thomas active filters, each representing one elementary proposition, tuned to a baseline frequency \(f_0\) [Section 17.2].
  2. Global Summing Bus: An OPA1612 summing amplifier configured to continuously calculate the spatial mean voltage \(V_{\text{mean}}(t) = \frac{1}{N} \sum_k V_k(t)\) and feed it back to all nodes [Section 17.3].
  3. Initialization Gate (\(S_{\text{init}}\)): An ADG1419 analog switch that routes an external sinusoidal reference signal \(V_{\text{init}}(t) = V_0 \sin(2\pi f_0 t)\) to the inputs of all resonators. When the control line Init Pulse is active, it locks all resonators to the same initial phase.
  4. Reset Switch Array (\(S_{\text{reset}}\)): A set of \(2N\) ADG1412 quad SPST analog switches. Each switch is placed in parallel with the \(1\text{ nF}\) integration capacitors (\(C_1, C_2\)) of each resonator. When the Reset Pulse is pulled high, these switches close, rapidly discharging the capacitors to \(0\text{ V}\) to clear the state of the register.
  5. Active Peak Detector Array: Each resonator output \(V_j(t)\) is connected to an active peak-hold circuit implemented with an OPA1611 and a low-leakage hold capacitor. This circuit tracks and stores the peak AC voltage amplitude achieved by each channel during the evolution phase.
  6. Multi-Channel Comparator Array: An array of low-offset analog comparators (e.g., TLV3501) that compares the DC outputs of the peak detectors to identify the channel with the highest voltage amplitude, outputting the target index as a digital signal.

Chapter 21: Circuit Analysis, Theoretical Complexity, and Speed Benchmarking

This chapter provides a full circuit analysis of the analog search register, verifying its mathematical complexity and calculating its practical execution speeds across different operational frequencies and database scales.

21.1 System Dynamics and Complexity Derivation

To analyze the circuit’s complexity, we model the system of \(N = 2^n\) coupled active resonators. Let the voltage of resonator \(j\) be \(V_j(t)\). The second-order differential equation governing each active node with coupling is:

\[\frac{d^2 V_j}{dt^2} + \frac{\omega_0}{Q} \frac{d V_j}{dt} + \omega_0^2 V_j = -\gamma_{\text{coup}} \omega_0^2 \left( V_j - V_{\text{mean}} \right) - \sigma_j(t)\]

where: * \(\omega_0 = 2\pi f_0\) is the natural resonant frequency of the uncoupled filter. * \(Q\) is the quality factor representing internal dissipation (for the Tow-Thomas loop, \(Q = R_D / R\)). * \(\gamma_{\text{coup}} = R / R_{\text{coup}}\) is the dimensionless coupling coefficient. * \(V_{\text{mean}}(t) = \frac{1}{N} \sum_{k=1}^N V_k(t)\) is the output of the global summing bus. * \(\sigma_j(t)\) represents the oracle’s phase-inversion term.

Matrix Formulation in the Proposition Space

Let the joint state vector of the propositional database be \(\mathbf{V}(t) = [V_1, V_2, \dots, V_N]^T \in \mathbb{R}^N\). Assuming a high quality factor (\(Q \to \infty\)) to simplify the analytical core, the system equations in matrix form are:

\[\frac{d^2 \mathbf{V}}{dt^2} + \omega_0^2 \left[ \mathbf{I}_N + \gamma_{\text{coup}} \left( \mathbf{I}_N - \frac{1}{N} \mathbf{J}_N \right) \right] \mathbf{V} = -\mathbf{\Sigma}(t)\]

where \(\mathbf{I}_N\) is the identity matrix, and \(\mathbf{J}_N\) is the all-ones matrix representing the projection operator \(|s\rangle\langle s| = \frac{1}{N}\mathbf{J}_N\) [Section 14.3].

Derivation of the \(\mathcal{O}(\sqrt{N})\) Search Speed

When the oracle is activated to target index \(t\), it flips the phase of the target’s coupling feedback, meaning \(\sigma_t(t)\) acts as a \(\pi\)-phase inverter. This splits the system’s eigenvalues into a symmetric bulk mode (where all non-target resonators oscillate in phase) and an asymmetric target mode.

The coupled system acts as two macroscopic oscillators exchanging energy. The envelope of the target node voltage \(A_{\text{target}}(t)\) grows according to the beat frequency \(\Omega_{\text{beat}}\) of this coupled system:

\[A_{\text{target}}(t) = V_0 \sin\left(\Omega_{\text{beat}} \, t\right)\]

The beat frequency is derived from the eigenvalue separation of the perturbed projection matrix:

\[\Omega_{\text{beat}} \approx \frac{\gamma_{\text{coup}} \omega_0}{2 \sqrt{N}}\]

The voltage amplitude of the target proposition reaches its maximum peak when the sine argument is \(\pi/2\):

\[\Omega_{\text{beat}} \, t_{\text{opt}} = \frac{\pi}{2} \implies t_{\text{opt}} = \frac{\pi}{2 \Omega_{\text{beat}}} = \frac{\pi \sqrt{N}}{\gamma_{\text{coup}} \omega_0}\]

Since \(\omega_0 = 2\pi f_0\):

\[t_{\text{opt}} = \frac{\sqrt{N}}{2 \gamma_{\text{coup}} f_0}\]

For a fixed coupling coefficient (e.g., \(\gamma_{\text{coup}} = 0.5\) as configured in Section 17.3), the optimal evolution time simplifies to:

\[t_{\text{opt}} = \frac{\sqrt{N}}{f_0}\]

This formal derivation demonstrates that the physical time required to isolate the target proposition scales as \(\mathcal{O}(\sqrt{N})\), verifying that the quadratic complexity scaling of Grover’s algorithm is preserved in this classical parallel analog wave-resonance space.

21.4 Component Tolerance and Matching Requirements

To prevent the search signal from splitting or dephasing, the individual resonant frequencies of the \(N\) physical channels must match within the resonance bandwidth:

\[\Delta f \le \frac{f_0}{Q}\]

  • For the Low-Frequency Configuration (\(f_0 = 10\text{ kHz}, Q \approx 63\)), the allowable mismatch is: \[\Delta f \le \frac{10\text{ kHz}}{63} \approx 158.7\text{ Hz} \implies 1.58\% \text{ frequency tolerance}\]
  • For the High-Frequency Configuration (\(f_0 = 10\text{ MHz}, Q \approx 50\)), the allowable mismatch is: \[\Delta f \le \frac{10\text{ MHz}}{50} \approx 200\text{ kHz} \implies 2.0\% \text{ frequency tolerance}\]

Since standard \(0.1\%\) thin-film metal resistors and \(1\%\) C0G ceramic capacitors are used [Section 17.1], the worst-case component-induced frequency drift is limited to:

\[\frac{\Delta f_{\text{actual}}}{f_0} \approx \frac{1}{2} \left( \frac{\Delta R}{R} + \frac{\Delta C}{C} \right) = \frac{1}{2} \left( 0.001 + 0.01 \right) = 0.55\%\]

Because \(0.55\% \ll 1.58\%\), the passive components maintain the frequency matching of the parallel channels well within the required limits of the coupled resonance, preventing dephasing across both configurations.
